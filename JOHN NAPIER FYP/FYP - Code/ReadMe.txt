- A unix executable of the CAD system is available in the “Executable” Folder
- The source code is contained in the folder "Final_Program"->"Final_Program"->"Final_Program"->"Main.cpp"
- In order to execute the code, OpenCV libraries need to be installed and available.
- Throughout the code, the directories need to be changed according to preference of the user:
	- The folder's required for the program to function are:
		- A 'test' folder containing the CT images
		- A 'filter' folder which will store the images after being processed by the noise removal filter
		- 4 different folders to store the resulting images from the segmentation process
